# sample_spell_dict
このリポジトリではオリジナルのspell check項目を定義できます。
`/cspell/.cspell.json`の
```json
    "words": [
        "yolo",
        "yolov",
        "streetsidesoftware",
        ...
```
内を更新することで使うことができるようになります。

This repository allows you to define your own spell check items.
in `/cspell/.cspell.json`
```json
    "words": [
        "yolo",
        "yolov",
        "streetsidesoftware",
        ...
```
You can use it by updating here.
